Небольшая инструкция

★ Смена знака производится посредством нажатия на кнопку " -/+ ".
Введите число, затем нажмите кнопку " -/+ " - число станет отрицательным. При повторном нажатии число станет положительным.
★ Извлечение квадратного корня из числа производится посредством кнопки " √ ".
Введите число из которого хотите извлечь квадратный корень, затем нажмите кнопку " √ " - на экране появится результат вычисления. При попытке извлечь корень из отрицательного числа появится сообщение об ошибке.
★ Возведение в степень производится посредством кнопки " ^ ".
Введите число которое хотите возвести в степень, затем нажмите кнопку " ^ " и введите саму степень. После нажатия " = " на экране появится результат вычисления.
Этот калькулятор не сможет вычислить число Грэма, но он вполне неплох!
